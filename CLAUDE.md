# Bonded Labs Website

## Overview
Marketing/landing site for Bonded Labs LLC — a solo software studio. Hosted on GitHub Pages with custom domain `bondedlabs.org`.

## Tech Stack
- Plain HTML + CSS, no build step, no JavaScript
- Hosted via GitHub Pages (deploy from `main` branch, root)
- Custom domain configured via `CNAME` file + Namecheap DNS
- SSL via GitHub/Let's Encrypt

## Structure
- `index.html` — main landing page with project cards
- `privacy.html` — privacy policy
- `terms.html` — terms of service
- `css/style.css` — single stylesheet, uses CSS custom properties for theming
- `img/` — favicon and logo/banner
- `CNAME` — custom domain config (`bondedlabs.org`)

## Key Details
- Contact email: `support@bondedlabs.org`
- Current projects listed: Fantasy Friend (live at fantasyfriend.online), Home Management App (incubating)
- Refund policy defers to individual product sites
- Favicon and logo are shared across all three pages
- Privacy/terms pages link back to home via the logo
